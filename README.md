# 🎮 Simon Says Game

An interactive and addictive memory-based game built using **HTML, CSS, and JavaScript**.  
Test your memory skills by following the color sequence generated by the game. Each level becomes harder as the sequence grows longer!

## 🧠 About the Game

Simon Says is a classic memory game where players must remember and repeat the exact sequence shown by the game.

The game is very simple and easy to play, but it quickly becomes highly addictive and challenging as the levels increase. It’s a fun brain-training game that improves memory, concentration, and reaction skills.
